bgee_calls {Bgee} R Documentation

Do bgee enrichment

Description

Usage

bgee_calls(bgee, geneSet,
    development.stage = "*");

Arguments

bgee

[as BgeeDiskReader]

geneSet

[as string]

development.stage

this function returns the enrichment result for all existed development stages. [as string]

Details

Authors

Erica

Value

the return data value is depends on the parameter value of `development_stage`. + for a specific development_stage id, this function just returns a enrichment result dataframe + for default value, a list that contains enrichment result for each development_stage will be returned

clr value class

  • EnrichmentResult

Examples

 let model = read.backgroundPack("./demo.dat");
 let gene_ids = ["gene_a","gene_b","gene_c","gene_d"];
 let enrich = as.data.frame(bgee_calls(geneSet = gene_ids));
 
 print(enrich);
 
 write.csv(enrich, file = "./bgee_calls.csv");
