spatial_annotations {singleCell} R Documentation

Create spatial annotations data set for each spot data

Description

Usage

spatial_annotations(x, y, label,
    colors = "paper",
    barcode = NULL);

Arguments

x

X of the spot dataset, a numeric vector

y

Y of the spot dataset, a numeric vector

label

A character vector that assign the class label of each spatial spot, the vector size of this parameter must be equals to the vector size of x and y.

colors

A character value of the spatial class color palette name or a vector of color code character for assign each spatial spot.

barcode

the raw reference barcode to the spots

env

[as Environment]

Details

Authors

Erica

Value

this function returns data object of type SpotAnnotation.
