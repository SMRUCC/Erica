read.TCGA_mtx {singleCell} R Documentation

Read expression matrix from a TCGA MTX dataset folder

Description

Usage

read.TCGA_mtx(dataset);

Arguments

dataset

a folder path to the TCGA MTX dataset files, this folder should includes the data files: 1. barcodes.tsv 2. features.tsv 3. matrix.mtx. [as string]

Details

Authors

Erica

Value

a clr DataFrame object that contains all information that read from the given TCGA dataset folder.
