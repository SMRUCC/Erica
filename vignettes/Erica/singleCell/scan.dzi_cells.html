scan.dzi_cells {singleCell} R Documentation

Scan all single cell shapes from the given dzi slide data

Description

Usage

scan.dzi_cells(dzi, level, dir,
    ostu.factor = 0.7,
    noise = 0.25,
    moran.knn = 32,
    flip = FALSE,
    split.blocks = FALSE,
    split.IHC1.channels = FALSE);

Arguments

dzi

metadata of the dzi image. [as DziImage]

level

usually be the max zoom level. [as integer]

dir

A directory path that contains the image files in current `level`. [as IFileSystemEnvironment]

ostu.factor

[as double]

noise

quantile level for filter the polygon shape points. all cell shapes which has its shape points less than this quantile level will be treated as noise. [as double]

moran.knn

[as integer]

split.blocks

[as boolean]
