phenograph {phenograph} R Documentation

Run PhenoGraph algorithm

Description

Usage

phenograph(x,
    y = NULL,
    k = 30,
    link.cutoff = 0,
    knn.cutoff = 0,
    score = NULL,
    subcomponents.filter = 0,
    knn2 = 16,
    joint.cutoff = 0,
    n.threads = 32);

Arguments

x

This parameter should be an expression matrix data object. [as Matrix]

k

The KNN parameter for the phenograph analysis on the input matrix `x`. [as integer]

link.cutoff

[as double]

subcomponents.filter

removes small subnetwork. [as integer]

Details

Authors

Erica

Value

this function returns data object of type NetworkGraph.

clr value class

  • NetworkGraph

Examples

 require(GCModeller);
 
 imports "geneExpression" from "phenotype_kit";
 
 let expr_mat = load.expr(file = "./expr.csv");
 let clusters = phenograph(expr_mat, k = 16);
 
 require(igraph);
 
 # get the cluster result
 print(as.data.frame(V(clusters)));
 print(as.data.frame(E(clusters)));
