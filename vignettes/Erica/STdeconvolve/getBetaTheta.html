getBetaTheta {STdeconvolve} R Documentation

get deconvolve result matrix

Description


Pull out cell-type proportions across pixels (theta) and
cell-type gene probabilities (beta) matrices from fitted
LDA models from fitLDA

Usage

getBetaTheta(LDA, corpus,
    top.genes = 25);

Arguments

LDA

an LDA model from `topicmodels`. From list of models returned by fitLDA. [as LdaGibbsSampler]

corpus

If corpus is NULL, then it will use the original corpus that the model was fitted to. Otherwise, compute deconvolved topics from this new corpus. Needs to be pixels x genes and nonnegative integer counts. Each row needs at least 1 nonzero entry (default: NULL). [as STCorpus]

top.genes

[as integer]

Details

Authors

Erica

Value

A Deconvolve object that contains

+ beta: cell-type (rows) by gene (columns) distribution matrix. Each row is a probability distribution of a cell-type expressing each gene in the corpus. + theta: pixel (rows) by cell-types (columns) distribution matrix. Each row is the cell-type composition for a given pixel.
