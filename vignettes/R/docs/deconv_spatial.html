deconv_spatial {Erica} R Documentation

Make spatial matrix deconvolution

Description

Reference-free cell-type deconvolution of pixel-resolution spatially resolved transcriptomics data

Usage

deconv_spatial(expr.mat,
    n.layers = 4,
    top.genes = 1000,
    alpha = 2,
    beta = 0.5,
    iteration = 150,
    prefix = 'class',
    make.gene.filters = TRUE,
    filter.range = [0.95],
    unify.scale = 10,
    log.norm = TRUE,
    n.threads = NULL);

Arguments

expr.mat

a dataframe object or a GCModeller expression matrix object, the matrix object should be in format of spatial spot in rows and the gene features in columns.

n.layers

an integer value for specific the cell layer class number

top.genes

set the number of top genes when export the expression matrix in each cell layers

n.threads

default null means controls the thread number for parallel sampling via the r environment config: getOption("n_threads").

Details

the generated expression matrix object could be save as csv matrix by use the function geneExpression::write.expr_matrix from the GCModeller package.

Authors

xieguigang

Value

A tuple list object that contains multiple result object inside:

  1. single_cells: an expression matrix object that contains the single cell data
  2. deconv_spatial: an expression matrix object that contains the cell

layer deconvolution result data

  1. cell_layers: the cell layer composition data
  2. gibbs_LDA: the result of the LDA gibbs sampling result outputs, the top

features in each cell class
