plot_pie {Erica} R Documentation

Plot the scatter pie data

Description

Usage

plot_pie(cell.layers,
    x = NULL,
    y = NULL);

Arguments

cell.layers

the cell class probablity matrix data

x

the x axis for draw a pie, could be x scan location of the spatial data or x dimension of the single cells umap embedding result.

y

the y axis for draw a pie, could be y scan location of the spatial data or y dimension of the single cells umap embedding result.

Details

Display the cell/spot class distribution

Authors

xieguigang

Value

This function has no value returns.
